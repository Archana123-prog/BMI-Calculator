## 🧮 How It Works

The BMI is calculated using the formula:

[
BMI = \frac{weight \ (lbs)}{height^2 \ (in)} \times 703
]

### 📊 BMI Categories:

| BMI Range | Status      |
| --------- | ----------- |
| < 25      | Underweight |
| 25 - 30   | Healthy     |
| > 30      | Overweight  |
